# diplo-net-project
Fifth module project

# Construsales
Desarrollo in-house que permita gestionar y dar seguimiento a las oportunidades de venta de clientes, utilizando tecnología .NET para cumplir con el objetivo y adaptarlo a un costo más accesible

# Funciones
	Gestionar y dar seguimiento a clientes y vendedores.

	Administrar los objetivos y oportunidades de venta.

	Motivar a los vendedores para buscar prospectos, abrir oportunidades y cerrarlas con éxito.

	Medir el desempeño de los vendedores.

	Mantener informados a los vendedores sobre noticias del sector y económicas del país que puedan ser relevantes para la búsqueda de nuevas oportunidades.
